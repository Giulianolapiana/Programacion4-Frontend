Programación IV: Desarrollo Frontend con React
¡Bienvenido al repositorio oficial de la cursada! Este espacio está dedicado al aprendizaje, experimentación y despliegue de aplicaciones modernas utilizando el ecosistema de React.js.

🚀 Tecnologías Principales
Este proyecto (o conjunto de prácticas) utiliza las siguientes herramientas:

React 18+ (Hooks, Context, Composition)

Vite (Build Tool ultra rápido)

Tailwind CSS / Styled Components (Para el estilado)

React Router Dom (Navegación SPA)

Axios / TanStack Query (Consumo de APIs)
